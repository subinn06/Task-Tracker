# Task Tracker

A simple full stack JavaScript web application for managing tasks and viewing smart insights about workload.  
This project demonstrates basic CRUD operations, database integration, and frontend backend communication using React and Express.

---

## Setup

Backend -

cd backend   npm install   node server.js

Frontend -

cd frontend   npm install   npm start

---

## Features

Task Management - 

- Create new tasks (with title, description, priority, due date)
- View all tasks
- Update task status (Open, In Progress, Done)
- Filter and sort tasks by priority, status, or due date

Smart Insights -

- Displays workload summary like:
  - Number of open tasks
  - Tasks due within 3 days
  - Most common task priority
- The insight string updates automatically when data changes.
